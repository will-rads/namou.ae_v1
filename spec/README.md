# Spec

This folder contains the product spec + UI reference screenshots for rebuilding the Adobe XD prototype as a real web app.

## Files

- `notes.md`  
  Jad’s requirements and guidance (what the app must become + key UX rules).

- `screens/`  
  Reference screenshots from the current prototype, in order:
  - `00-landing-closed.png`
  - `01-landing-area-picker-open.png`
  - `02-rak-landscape-map.png`
  - `03-rak-landscape-map-gallery-overlay.png`
  - `04-rak-landscape-map-gallery-overlay-al-hamra.png`
  - `05-master-plan-overview.png`
  - `06-asset-specifications.png`
  - `07-gallery-images.png`
  - `08-site-visit-itinerary.png`

## How to use (Claude Code workflow)

1) Read `notes.md` first (requirements + constraints).
2) Use `screens/` to recreate layout, spacing, typography, and component structure.
3) Build the UI in small steps:
   - scaffold app + global layout
   - implement left nav + page shells
   - implement each page to match screenshots
   - then apply Jad’s changes as tasks

## Notes

- The Adobe XD share link is treated as reference only.
- Screenshots are the source of truth for UI layout and styling.
