# Brand Quick Reference (Namou)

## Tone
Strategic, direct, professional. Avoid fluff. Use hard data, percentages, and clear calls to action.

## Typography
Primary: Bahnschrift (headlines, tagline, ROI callouts)
Secondary: Graphik Web

## Colors
Forest Green: #003D2E  
Deep Forest: #273D37  
Mint Green: #87c6ae  
Mint White: #f7fbf8  

## Logo rules
Respect clear space around logo (minimum clear zone equal to height of the “N” on all sides).
Do not stretch/distort. Do not change Arabic script color. Don’t place on busy backgrounds.
