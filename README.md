# namou.ae

Work repo for the Namou web experience.

## What this repo is for
- Rebuilding the current Adobe XD prototype into a real, maintainable web app
- Implementing Jad’s requirements and future iterations as code

## Specs / Design Reference
All product requirements and UI screenshots live here:
- `spec/notes.md` (Jad’s requirements)
- `spec/screens/` (reference screenshots)
- `spec/README.md` (how to use the spec)

## Working Agreement
- Keep commits small and descriptive
- Prefer clean, boring code over clever code
- Don’t “design new stuff” unless it’s explicitly requested in `spec/notes.md`

## Getting started (dev)
> Fill this in once the stack is scaffolded.

- Install dependencies: `npm install`
- Run dev server: `npm run dev`
